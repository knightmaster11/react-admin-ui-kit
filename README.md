## рџ›  Installation

Clone repository

```bash
git clone https://github.com/knightmaster11/react-admin-ui-kit.git
```

Go to project directory

```bash
cd react-admin-ui-kit
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open browser

```text
http://localhost:5173
```
