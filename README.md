# React Admin UI Kit

Modern Admin Dashboard UI built with React, Vite, and Tailwind CSS.

## 🚀 Tech Stack

- React 18
- Vite 6
- JavaScript + SWC
- Tailwind CSS 3
- React Router DOM

## 📁 Project Structure

```text
src
├── components
├── layouts
├── pages
├── routes
├── services
├── hooks
├── utils
├── styles
└── App.jsx
```

## 🛠 Installation

```bash
git clone https://github.com/knightmaster11/react-admin-ui-kit.git
cd react-admin-ui-kit

npm install
npm run dev
```
